# IM920c/s/sL USB Interface Board V2
インタープラン株式会社のの無線モジュール、IM920c/s/sLシリーズをUSB接続するためのインターフェースボードです。\
純正品と異なる点として通電確認用LED、UARTのTx,Rxのインジケータを追加しています。\
※LEDはUARTバスに直結のため、未検証ですがボーレートを上げると信号が乱れる可能性があります。修正を検討中です。
